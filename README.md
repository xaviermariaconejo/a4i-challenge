# React + TypeScript + TailwindCSS + Vite

### Installation
```bash
npm run install:all
```

### To try this project

```bash
npm run start
```
And then check on: http://localhost:4175/

A different way is to execute, in one terminal
```bash
npm run backend:dev
```

And in other terminal
```bash
npm run build && npm run preview
```

And then check on: http://localhost:4175/

### To execute tests
```bash
npm run test
```
